--- ADDED ---

- Added new providers
-- Pirate Weather (API key required)
-- Visual Crossing (API key required)
-- Tomorrow IO (API key required)
-- StormGlass (API key required)
-- Weatherbit (API key required)
-- QWeather (API key required)

- Improved alerts support
-- "NWS" → NOAA NWS (USA Only, No API Key required)
-- Improved "MET Norway" → MET Norway (Norway only, No API Key required)
-- "MeteoAlarm" → EUMETNET MeteoAlarm (Most Europe countries, No API Key required)

- Alerts from providers are supported, if you use the following providers as well
-- Pirate Weather (API Key Required)
-- Visual Crossing (API Key required)
-- "Weather API" (API Key required)

- Added support for multi-location support
- Added support to save locations and switch them in the "Locations" menu or the widget
- Added support for weather tray icon
- Added support to choose manually location using a map (Powered by OpenStreetMap)
- Added support to refresh or choose a location with a right click in the widget.
- Added support to show/hide forecast and details in the widget
- Added ability to change the location name

- Fix a critical bug where if you choose another provider unless open-meteo or adaptive mode enabled, you cannot find any location manually
- Fix 18N_ARGUMENT_MISSING in some dialogs
- Fix a bug where if you control the widget height/weight it was affected in the widget panel space
